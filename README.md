# Pixel Art
Pixel art became a passion of mine a few years ago, when I created some small pixel art games on Scratch. 
Ever since, I've loved creating pixel art and sharing it with others. I use my pixel art for games, websites, portfolio work,
and as a way to relax. 

I do all of my pixel art on [Piskel](https://www.piskelapp.com/). Check it out and make something cool!
